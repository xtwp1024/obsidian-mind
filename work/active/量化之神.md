## TimesFM 集成

**TimesFM 2.5** (Google, 200M参数) 已集成到信号系统。

### 核心文件

- 信号生成器: `quant_trading/signal/timesfm_generator.py`
- 测试脚本: `quant_trading/experiments/test_timesfm.py`

### 使用方法

```python
from quant_trading.signal import TimesFMGenerator

gen = TimesFMGenerator(
    symbol="BTC/USDT",
    horizon=24,           # 预测步数
    context_len=256,      # 上下文长度（max 1024）
    buy_threshold=0.01,   # 预测涨幅>1%买入
    sell_threshold=-0.01, # 预测跌幅<-1%卖出
    use_quantile=True,    # 使用分位数预测（更稳健）
)
signals = gen.generate(df)
```

### 信号逻辑

- **BUY**: 预测收益 > buy_threshold
- **SELL**: 预测收益 < sell_threshold
- **HOLD**: 其他

信号 strength = min(|pred_return| / threshold, 1.0)

### TimesFM 模型

- 模型: `google/timesfm-2.5-200m-pytorch`
- 参数: 200M
- Context: 最长 16k
- 量化预测: P10-P90 分位数头

### GainsBot 信号系统 (2026-04-10)

**新增模块**: `quant_trading/signal/gainsbot_generator.py`

**集成模型**:
| 模型 | 功能 | 信号 |
|------|------|------|
| Kelly Criterion | 凯利准则 - 最优仓位计算 | LONG/SHORT/NEUTRAL |
| Omega Divergence | 欧米伽背离 - 动量分析 | BULLISH/BEARISH/NEUTRAL |
| Hilbert Transform | 希尔伯特变换 - 周期/相位 | BUY/SELL/HOLD |
| Bayesian Posterior | 贝叶斯后验 - 概率方向 | BUY/SELL/HOLD |
| Shannon Entropy | 香农熵 - 市场不确定性 | TREND/CHAOS/NEUTRAL |
| Black-Scholes | 布莱克-舒尔斯 - 期权/风险 | BUY/SELL/HOLD |
