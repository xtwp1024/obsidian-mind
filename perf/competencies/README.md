# Competencies

This folder holds one note per competency from your organization's competency framework.

## How it works

1. **Create** a competency note using the `Competency Note` template
2. **Link** work notes to competencies in their `## Related` section
3. **Evidence accumulates via backlinks** -- open any competency note and check the backlinks panel to see all work that demonstrates it
4. **Brag Doc** aggregates evidence per quarter in `#### Competency Evidence` sections
5. **Review Template** structures self-assessment around competencies

## Frontmatter fields

- `current-level` -- your assessed level for this competency
- `target-level` -- where you're aiming next

## Tips

- Don't maintain evidence lists manually in competency notes -- let backlinks do the work
- Use annotations when linking: `[[Competency Name]] -- what was demonstrated`
- Review competency backlinks when preparing for performance reviews
